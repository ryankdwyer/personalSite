Cohearence is a web app coupled with a Chrome Extension. The Chrome Extension seamlessly tracks your music listening from several supported sites. Then, via the Chrome Extension or web app, you can playback any song you have previously listened to without having to go back to that site.

Check out a quick demo of it here.

exAudio is built on electronJS as a stand alone application. It allows you to playback all common audio codecs and several lossless audio codecs, including FLAC. This is especially useful if you don't want iTunes to convert your lossless files.

Download it here.

Remote-Code allows two people to code on a shared editor while video chatting. You can download and upload files to share as well. Remote-Code also provides a REPL console for quick testing.

Give it a try here.
